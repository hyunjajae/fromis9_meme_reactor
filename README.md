# fromis_9 Meme Reactor (프로미스나인 밈 리액터) 🎬

웹캠으로 **손동작 + 표정**을 인식해서, 정해진 포즈가 잡히면
해당 프로미스나인 밈 영상을 옆 창에 띄워주는 파이썬 프로그램입니다.

전신 포즈(MediaPipe Pose)는 쓰지 않고, **손(MediaPipe Hands) + 얼굴(FaceMesh) + 표정(FER)** 만으로 동작합니다.

## 2. 설치 방법 (한 줄씩 따라 하세요)

> 명령어는 **PowerShell**(윈도우 시작 → "PowerShell" 검색)에서 실행합니다.

### (1) 프로젝트 폴더로 이동
```powershell
cd $HOME\Desktop\fromis9_meme_reactor
```
- `cd` = change directory(폴더 이동). `$HOME`은 내 사용자 폴더(C:\Users\samsung)를 뜻합니다.

### (2) 가상환경 만들기 (선택이지만 추천)
```powershell
python -m venv venv
```
- `venv`라는 독립된 파이썬 공간을 만듭니다. 패키지가 다른 프로젝트와 섞이지 않아요.

### (3) 가상환경 켜기
```powershell
.\venv\Scripts\Activate.ps1
```
- 성공하면 줄 앞에 `(venv)` 가 붙습니다.
- 만약 "실행 정책" 오류가 나면 아래를 한 번 실행한 뒤 다시 켜세요:
  ```powershell
  Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
  ```

### (4) 패키지 설치
```powershell
pip install -r requirements.txt
```
- `requirements.txt`에 적힌 패키지를 한 번에 설치합니다. (TensorFlow 때문에 몇 분 걸릴 수 있어요)

### (5) 밈 영상 넣기
`memes` 폴더에 6개 mp4 파일을 위 이름 그대로 넣어주세요.

## 4. 인식하는 6개 포즈

| 우선순위 | 밈 | 포즈 |
|:---:|---|---|
| 1 | 이채영_재채기 | 한 손으로 입·코(얼굴 하단)를 가림 |
| 2 | 송하영_수플렉스 | 양손 주먹 + 두 손을 머리 위로 |
| 3 | 박지원_자동차를_몰거예요 | 양손 주먹 + 가슴 높이에서 핸들 잡듯 앞으로 |
| 4 | 박지원_마즈나요 | 양손을 가슴 앞에 가까이 모음 (주먹 아니어도 됨) |
| 5 | 송하영_그만말해인제 | 입을 크게 벌림 (손과 무관) |
| 6 | 이나경_날놔라집사 | 손이 화면에 안 보이고 + 찡그린 표정(angry/sad/disgust) |

> 위에서부터 검사해서 **처음 매칭되는 하나만** 재생합니다.
> 한 번 밈이 뜨면 영상이 **끝까지** 재생되고, 그동안은 새 인식을 잠시 멈춥니다(쿨다운).

## 5. 인식이 잘 안 될 때 — 숫자 튜닝하기

`main.py` 맨 위의 **튜닝 상수**만 바꾸면 됩니다.

| 상수 | 기본값 | 설명 |
|---|:---:|---|
| `STABLE_FRAMES` | 5 | 포즈를 몇 프레임 유지해야 확정할지. 너무 예민하면 ↑ |
| `MOUTH_OPEN_RATIO` | 0.55 | 입 벌림 인식 기준. 입을 조금만 벌려도 잡히게 하려면 ↓ |
| `WRIST_CLOSE_DIST` | 0.18 | 마즈나요(양손 모음) 인식 거리. 더 넓게 잡으려면 ↑ |
| `FACE_BAND` | 0.15 | 마즈나요 손 높이 허용 범위 |
| `FER_INTERVAL` | 5 | 표정 인식 주기(프레임). 느리면 ↑ (성능↑, 반응↓) |
| `CAM_INDEX` | 0 | 웹캠이 안 잡히면 1, 2로 바꿔보세요 |
| `MEME_WIDTH` | 480 | 밈 영상 창 가로 크기(px) |

---

## 6. 자주 나는 오류

- **웹캠이 안 열려요** → `CAM_INDEX`를 0 → 1 → 2 로 바꿔보세요.
- **한글이 ??? 로 깨져요** → 윈도우라면 보통 자동으로 잡힙니다. 안 되면 `FONT_PATH`를 다른 한글 폰트 경로로 바꾸세요.
- **`No module named ...`** → 가상환경 `(venv)`이 켜져 있는지, `pip install -r requirements.txt`를 했는지 확인하세요.
- **TensorFlow 설치 충돌** → Python 3.10 또는 3.11 환경에서 설치하는 걸 추천합니다.
- **영상에 소리가 없어요** → OpenCV는 영상만 재생하고 소리는 재생하지 않습니다(스펙상 정상).

## 7. 동작 원리 한눈에 보기

```
웹캠 프레임
   ├─ MediaPipe Hands  → 손 21개 좌표 (주먹/손높이/손목거리 계산)
   ├─ MediaPipe FaceMesh → 코 위치 / 얼굴 가로폭 / 입 벌림 비율
   └─ FER (5프레임마다) → 표정 분류
            │
            ▼
   detect_meme(face, hands, emotion)  ← 우선순위대로 검사
            │
            ▼
   같은 결과 5프레임 연속 → 확정 → 밈 영상 재생(끝까지, 그동안 인식 멈춤)
```
